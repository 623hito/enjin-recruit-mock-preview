Enjin 採用LP（合作版 v7 / TOPページ）— オフライン版バンドル
=========================================================

■ 使い方
index.html をダブルクリック（ブラウザで開く）だけで表示できます。
インターネット接続は不要です。CSS・JS・フォント・画像をすべて同梱しています。

■ 中身
  index.html          TOPページ本体（CSS/JSはHTML内にインライン）
  assets/tailwind.js  Tailwind CSS（CDN版を同梱）
  assets/fonts.css    Webフォント定義（ローカル参照に書き換え済み）
  assets/fonts/       Alumni Sans / Noto Sans JP / Noto Serif JP（253ファイル）
  images/             掲載画像14点（実写素材・ロゴ）

■ 本バンドルは「TOPページ単体」です
TOPページの表示に必要なCSS・JS・フォント・画像はすべて含まれています。
ただし、以下の下層ページへのリンクは含まれていないためクリックしても開きません
（リンク切れになります）:
  会社を知る / 採用情報 / 働く環境 / ひとを知る / 営業職 /
  お知らせ一覧・記事 / インタビュー個別 / 職種詳細
→ 全ページ入りのバンドルが必要な場合はお申し付けください。

■ 注意
・本モックは内部レビュー用です。文言・数値・リンクの一部は「仮」の状態です。
・「コンセプトエリア 準備中」「◯◯仮」などの表示は、確定前を示す目印です。
・エントリー／説明会申込・SNS・360VRの各リンクはダミーです。
・写真素材は現行採用サイトおよびライセンス取得済みストック素材を使用しています。
